# MESSAGES

This set of endpoints allows you send messages, campaigns, statistics and status.

### SEND MESSAGES

This endpoint allows you to curate and send messages.

<Accordion
  items={sendMessageAPI}
  parameters={true}
  endpoint={"https://squadbygtco.com:8080/dashboard/message"}
  method={"POST"}
  header={"This API create and send Messages"}
/>

#### Sample Request

```js
{
    "is_campaign": true, 
    "reference": "abc_123",
    "use_bucket": true,
    "recipients": "1", 
    "use_template": true,
    "body": "1", 
    "sender": "GTBank"
}
```
